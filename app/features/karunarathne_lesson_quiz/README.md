# Lesson Player & Audio Quiz Engine
**Contributor:** Karunarathne K.P.P.T (IT22926876)

## Scope
This module implements:
- Audio-first lesson playback for visually impaired learners
- Hybrid TTS + offline audio support
- Gesture-based navigation
- Audio-based quiz engine with adaptive feedback

## Key Research Contributions
- Reinforcement-learning inspired question repetition
- Fully non-visual quiz interaction via gestures
- Offline-ready spoken learning pipeline
